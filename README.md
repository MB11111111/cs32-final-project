# Lifestyle Health Tracker

## Project Description
This project explores how lifestyle factors like  diet, exercise, and sleep relate to health outcomes. Using publicly available datasets, the project applies computational analysis to identify patterns and correlations between daily habits and indicators like BMI, blood pressure, and wellness.

## Computational Tasks
- **Data acquisition:** Accessing public datasets containing lifestyle and health information.
- **Data cleaning and preprocessing:** Handling missing data, normalizing values, and organizing variables.
- **Feature selection:** Identifying which lifestyle and health outcome variables to analyze.
- **Exploratory data analysis (EDA):** Visualizing trends and computing correlations between lifestyle factors and health outcomes.
- **Statistical modeling / predictive analysis:** Using regression or simple machine learning models to explore relationships.
- **Visualization and reporting:** Creating plots and summaries to communicate findings effectively.

## Tools
- Python  

## Goals
- Understand the impact of lifestyle choices on health.
- Practice computational data analysis and visualization.
- Produce clear insights that could inform personal or public health decisions.
