# Lifestyle Tracker Project

## Description
This project is a lifestyle tracker that analyzes relationships between exercise, sleep, and BMI.

The program:
- Loads a dataset of individuals
- Cleans missing data
- Calculates summary statistics
- Computes correlations between lifestyle factors and BMI
- Generates visualizations
- Provides interpretations of results for users

This makes the data easier to understand for someone without a statistics background.

---

## Features
- Handles missing data safely
- Analyzes:
  - Exercise vs BMI
  - Sleep vs BMI
- Generates scatterplots
- Interprets results in plain English
